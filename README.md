# 🎮 Python Taboo Game

Bu proje, klasik Taboo oyununun Python terminali üzerinde çalışan bir versiyonudur. Bilgisayar mühendisliği eğitimim sürecinde, Python'daki temel veri yapılarını ve oyun mantığını pekiştirmek amacıyla geliştirilmiştir.

## ✨ Özellikler
* **Geniş Kelime Havuzu:** İçerisinde birçok farklı kelime ve yasaklı kelime barındırır.
* **Puanlama Sistemi:** Doğru ve yanlış cevaplara göre anlık skor takibi yapar.
* **Kullanıcı Dostu Arayüz:** Terminal üzerinden basit ve anlaşılır komutlarla oynanabilir.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python 3.x
* **Mantık:** Listeler, Sözlükler (Dictionaries) ve Random modülü.

## 🚀 Nasıl Çalıştırılır?
1. Bilgisayarınızda Python'ın yüklü olduğundan emin olun.
2. Bu repository'i bilgisayarınıza indirin.
3. Terminali açın ve projenin olduğu klasöre gidin.
4. Aşağıdaki komutu çalıştırın:
   ```bash
   python taboo.py
